# ラオス語単語クイズ 設定手順書

このアプリで回答データをGoogleスプレッドシートに転送するための設定手順です。

## 1. Googleスプレッドシートの準備
1. [Googleスプレッドシート](https://sheets.new/)を新規作成します。
2. シートの名前を「ラオス語単語クイズ回答データ」などに変更します。

## 2. Google Apps Script (GAS) の設定
1. スプレッドシートのメニューから **「拡張機能」 > 「Apps Script」** を開きます。
2. エディタに最初からあるコードをすべて消去し、同梱の `gas_code.js` の内容をすべて貼り付けます。
3. 左上の「プロジェクト名」を適当な名前（例: QuizReceiver）に変更し、保存（フロッピーアイコン）します。

## 3. ウェブアプリとして公開（デプロイ）
1. 右上の **「デプロイ」 > 「新しいデプロイ」** をクリックします。
2. 種類の選択（歯車アイコン）で **「ウェブアプリ」** を選択します。
3. 設定を以下のようにします：
   - **次のユーザーとして実行**: 「自分」
   - **アクセスできるユーザー**: **「全員」** （※ここを「全員」にしないとデータが届きません）
4. 「デプロイ」をクリックします。
5. 承認を求められた場合は、自分のアカウントを選択し、「詳細」>「（プロジェクト名）に移動」>「許可」を押します。
6. 表示された **「ウェブアプリのURL」** をコピーします。

## 4. HTMLへのURL反映
1. `index.html` をテキストエディタで開きます。
2. 120行目付近にある以下の部分を探します：
   ```javascript
   const GAS_URL = "ここにGASのウェブアプリURLを貼り付けてください";
   ```
3. `"` の間に、先ほどコピーしたURLを貼り付けて保存します。

## 5. GitHubへの反映
1. 修正した `index.html` をGitHubにアップロード（プッシュ）してください。

これで、クイズ終了時に自動的にスプレッドシートへデータが送信されるようになります。
